Вопросы для зачета по предмету "Современные методы и средства разработки программного обеспечения"
https://raw.githubusercontent.com/Kondusov/pi-242/refs/heads/master/exam/questTOzachet.txt
Пример практического задания для зачета - https://raw.githubusercontent.com/Kondusov/pi-242/refs/heads/master/exam/zachet_zadanie.txt

Экзамен будет приниматься в письменном виде, для группы (два варианта) будет поставлена задача в форме технического задания, на написание программы/развертывания-конфигурирования-запуска приложения/сервиса  (на ЯП JS,PHP,БазаДанных,CMS и другие технологии из учебной программы), в программе предположительно будут использованны функции, массивы, условия, циклы.
Время отведенное на выполнение задания - 40 минут.
После передачи файлов программы на проверку, студент должен устно ответить на 3 вопроса преподавателя, касающихся его выполненной работы, либо общих вопросов по дисциплине.
Экзамен будет считаться сданным в случае выполнения студентом, не менее 75% технического задания, и не менее двух верных ответов в устной форме.

На экзамене запрещается:
-пользоваться смартфоном и мобильным телефоном
-смотреть в мониторы других студентов
-обсуждать способы выполнения задания
-пользоваться поиском в сети интернет (только официальная документация)
(в случае нарушения этих правил - работа студента не принимается к проверке и выставляется оценка -"1").

Условия допуска к экзамену:
- сданы все домашние задания
- сданы все контрольные и самостоятельные работы
- студент пришел вовремя (не позднее начала зачета)

Вопросы к зачету (теория).

1. Доска задач и Agile-методология разработки программного обеспечения (как организованна доска).
2. GIT — cистема контроля версий (в чём польза и удобство. назовите основные операции).
3. Языки программирования для web разработки (для каких задач - конкретный язык или фреймворк).
4. Верстка веб-страниц на языке HTML (какие элементы знаете, структура страницы).
5. Клиентские технологии программирования на примере языка программирования JavaScript (возможности,
основные конструкции языка).
6. Объектно-ориентированное программирование (объекты, классы, интерфейсы, трейты, свойства, методы, наследование, статические методы, абстракции).
7. Серверное программирование (Back-end, MySQL, phpMyAdmin, Web сервер Apache, DNS записи ).
8. CMS - система управления контентом (CMS wordpress - для чего, какие возможности, архитектура).
